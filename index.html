React Shop

2023.6(1인 개인 프로젝트)

  • 전역상태관리 라이브러리로 Redux Tool kit을 활용하여 Redux를 사용해보면서 Context Api를 사용했을때에는 처리하기 힘든 부분도 Reducer를 이용하여 쉽게 처리 하고 리랜더링을 할 수 있는 장점을 느낄 수 있었습니다.
  • css 프레임워크로 tailWind css를 사용하여 따로 css 파일을 만들거나 css-in-js 형식을 사용하여 개발하는 것 보다 확실히 빠르다는 것을 느낄 수 있는 계기가 되었습니다.
  • Swiper 라이브러리를 적용하여 간단하게 swipe를 적용하고 옵션을 설정 하여 오래걸릴 수 있는 슬라이드를 빠르게 제작 할 수 있었습니다.
  • TypeScript를 사용하여 추후 생길 수 있는 타입에 관한 오류를 미리 제거하며 타입에 대한 정보를 미리 지정하여 자동완성을 사용 할 수 있는 편리함을 알게되었던 프로젝트입니다.

  • 처음으로 git을 이용해 협업이라는걸 해보고 프론트엔드 팀장을 맡아 프로젝트를 진행하며 많은 점을 알 수 있었던 프로젝트였습니다.
  • 원래 다른 팀원의 파트인 SSE 연결을 맡으면서 처음 사용하게 된 event-source-polyfill을 사용하여 연결해보고 연결과정에서 생기는 오류를 해결하는 경험을 가질 수 있었습니다

  • 프로젝트의 특성상 모바일에서 보다 많이 사용될 수 있는 주제이기 때문에 PWA 를 이용한 웹앱 제작의 경험을 해볼 수 있었습니다.
